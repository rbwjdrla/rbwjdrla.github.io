HTML 기초 다지기: 웹 페이지의 뼈대

작성일: 2025년 6월 11일

1. HTML이란 무엇인가?

HTML (HyperText Markup Language)은 웹 페이지의 구조를 정의하는 마크업 언어입니다. 우리가 보는 모든 웹 페이지는 HTML로 이루어져 있습니다.

텍스트, 이미지, 링크 등 다양한 요소를 웹 페이지에 배치하고 구조화하는 역할을 합니다.

2. 기본 문서 구조

모든 HTML 문서는 다음과 같은 기본 구조를 가집니다.


<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>페이지 제목</title>
</head>
<body>
    <!-- 여기에 콘텐츠를 작성합니다 -->
</body>
</html>
  • <!DOCTYPE html>: 문서 유형을 HTML5로 선언합니다.
  • <html>: HTML 문서의 시작과 끝을 나타냅니다. `lang="ko"`는 한국어 문서임을 의미합니다.
  • <head>: 웹 페이지의 메타데이터(문서 정보)를 포함합니다. 사용자에게 직접 보이지 않습니다.
  • <body>: 웹 페이지의 실제 내용(콘텐츠)이 들어가는 부분입니다. 사용자에게 보이는 모든 요소가 여기에 작성됩니다.

3. 자주 사용하는 HTML 태그

가장 기본적이고 자주 사용되는 태그 몇 가지를 정리해 보았습니다.

  • <h1> ~ <h6>: 제목 태그 (숫자가 작을수록 중요한 제목)
  • <p>: 단락 (Paragraph) 태그
  • <a href="링크 주소">: 하이퍼링크 (Anchor) 태그
  • <img src="이미지 경로" alt="이미지 설명">: 이미지 태그
  • <ul> / <ol> / <li>: 순서 없는 목록 / 순서 있는 목록 / 목록 아이템
  • <div>: 의미 없는 일반적인 컨테이너 (레이아웃 구성에 자주 사용)
  • <span>: 의미 없는 인라인 컨테이너 (텍스트 일부에 스타일 적용)

더 많은 태그와 속성은 MDN Web Docs를 참고하세요.
